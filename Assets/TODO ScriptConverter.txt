LIMITATION :

JS > C#

- commentaires entre fin finction declaration et bracket d'ouverture



- variable declaration, without type definition and the value comes from a function or a class without "new" keyword :
var test = DoSomething();
Can't guess if DoSomething() is a function or a class, can't guess the type => nothing gets converted

- var declaration in a for/foreach loop, can't guess the variable type   for (var name in array) > foreach (?? name in array)
=> nothing is done

- does not add the IEnumerator return type to functions that yields.


- single quotation marks not converted to double quotation marks in string[] array

- some leftover String types (function return type)
